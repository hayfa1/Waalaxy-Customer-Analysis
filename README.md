## Installing
use virtual environment python: 
```bash
python -m venv yourvenv
```
Activate virtual environment:
```bash
yourvenv\Scripts\activate
```
update pip:
```bash
pip install --upgrade pip
```
install requirements:
```bash
pip3 install -r requirements.txt
```
